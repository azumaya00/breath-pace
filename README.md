# Breath Pace

個人用・非医療目的の呼吸ペースメーカー Web アプリケーション

## ⚠️ 重要な注意事項

**本アプリケーションは非医療目的の個人用ツールです。**

- 医療行為や治療を目的としたものではありません
- 自己責任でご使用ください
- 体調不良や持病がある場合は、医師にご相談の上でご使用ください
- 画面スリープ時にタイマーが停止する制約があります

## 機能

- **7種類の呼吸プリセット**
  - **内養功系**
    - 内養功・基礎（4-4-4）
    - 内養功・深める（7-7-7）
  - **日常調整系**
    - 日常調整（4-0-6）
    - 夜の調整（4-0-8）
    - ニュートラル（5-0-5）
    - 朝の調整（6-0-4）
    - 緊張解放（3-0-6）

- **直感的なUI**
  - 体調・目的に合わせたプリセット選択
  - リアルタイムのフェーズ表示
  - 残り秒数とサイクル数の表示
  - 回数選択と所要時間の表示
  - 戻るボタンでプリセット選択に戻れる

- **音声フィードバック**
  - 1秒ごとのカウント音
  - フェーズ切替音

- **多言語対応**
  - 日本語 / 英語
  - 言語切替ボタンで切り替え可能
  - 選択は保存され、次回アクセス時も維持

- **テーマ切替**
  - ライト / ダーク / システム（自動）
  - テーマ切替ボタンで切り替え可能
  - 選択は保存され、次回アクセス時も維持

## 技術スタック

- **SvelteKit** - フレームワーク
- **TypeScript** - 型安全性
- **Vite** - ビルドツール
- **Vitest** - テストフレームワーク
- **Web Audio API** - 音声生成

## 開発

### セットアップ

```bash
pnpm install
```

### 開発サーバー起動

```bash
pnpm dev
```

### ビルド

```bash
pnpm build
```

### プレビュー

```bash
pnpm preview
```

### テスト

```bash
pnpm test
```

### 型チェック

```bash
pnpm check
```

## プロジェクト構造

```
src/
├── lib/
│   ├── components/     # UIコンポーネント
│   ├── types.ts        # 型定義
│   ├── presets.ts      # プリセットデータ
│   ├── timer.ts        # タイマーロジック
│   ├── audio.ts        # 音声生成
│   └── i18n.ts         # 多言語対応
├── routes/
│   └── +page.svelte    # メインページ
└── app.css             # グローバルスタイル
```

## 設計思想

- **状態管理**: 明確な state machine による状態管理
- **テスト可能性**: ロジックとUIの分離により、テスト可能な設計
- **型安全性**: TypeScriptによる型定義
- **アクセシビリティ**: セマンティックなHTMLと適切なコントラスト
